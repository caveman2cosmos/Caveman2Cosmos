
BTSFlagMod
-------------------


This Mod changes the flags for all of the Civilizations in Beyond The Sword:


To use-

Extract the file in the directory:    Documents\My Games\Beyond the Sword\MODS 


* Do not copy files to your C: drive directory where you installed
the game. Make sure you copy to your My Games folder only!


Thanks to Bad Ronald as I used his mod/dds files as a starting point!!
